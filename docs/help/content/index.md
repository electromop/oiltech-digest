# OilTech Digest Docs

Документация работает как отдельный Docker-сервис на **MkDocs Material**. Основное приложение остается на FastAPI, а `/help` проксируется в docs-сервис через Caddy.

![Основной экран OilTech Digest](assets/signals-overview.png)

## Разделы

| Раздел | Для кого | Что внутри |
|---|---|---|
| [Для пользователей](user/index.md) | редактор, аналитик, руководитель направления | работа с сигналами, источниками, тегами, скорингом и дайджестом |
| [Для разработчиков](developer/index.md) | backend/frontend/devops | архитектура, API, БД, фоновые задачи, external worker, деплой |
